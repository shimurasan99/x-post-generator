# 投資系X投稿文生成ツール

ゴールド、BTC、仮想通貨、為替、投資知識、EA・自動売買、LINE誘導、話題ニュースに関するX投稿文を作るStreamlitアプリです。

X APIは使いません。自動投稿もしません。投稿文を5個作り、コピーして使える初心者向けツールです。

## できること

- 投稿テーマを入力できます
- 投稿テーマをプルダウンから選べます
- 最新ニュースをRSSから取得してテーマ候補にできます
- 投稿ジャンルを選択できます
- 投稿トーンを「煽り強め」「ポップ強め」「落ち着きめ」から選べます
- 日本語の投稿文を5個生成します
- 文字数は140〜280字に収めます
- 禁止表現を避けます
- 生成結果をCSV形式テキストとしてコピーできます
- 投稿テーマに合わせた4コマ漫画作成プロンプトを生成できます
- 確認用の簡易4コマPNGも任意で生成できます
- 投稿文にはXで読みやすい改行を自動で入れます

## 禁止している表現

以下のような強い断定や買い煽り表現は使わないようにしています。

- 絶対稼げる
- 必ず上がる
- 今すぐ買え
- 確実に儲かる
- 必ず
- 絶対
- 確実
- 断言
- 間違いない
- 100%

## 必要なもの

- Python 3.10以上
- インターネット接続

## 使い方

### 1. 仮想環境を作る

ターミナルでこのフォルダを開き、次のコマンドを実行します。

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2. ライブラリをインストールする

```bash
pip install -r requirements.txt
```

### 3. アプリを起動する

```bash
streamlit run app.py
```

### 4. ブラウザで使う

起動するとブラウザに画面が表示されます。

1. 左側のメニューで投稿ジャンルを選びます
2. テーマの選び方を選びます
3. 最新ニュース、テーマ候補、または手入力で投稿テーマを決めます
4. 投稿トーンを選びます
5. 4コマ漫画用プロンプトを作成するか選びます
6. 「投稿文を5個生成する」を押します
7. 生成された投稿文と4コマ漫画作成プロンプトを確認します
8. 投稿文、CSV形式テキスト、4コマ漫画作成プロンプトをコピーできます
9. プロンプトをChatGPTの画像生成に貼り付けます

## 最新ニュース取得について

「最新ニュースから選ぶ」を選択すると、ジャンルに合うキーワードでGoogleニュースRSSを取得します。

取得するニュースは、公開日時が直近2日以内の記事だけに絞っています。

APIキーは不要です。ただし、ニュース取得にはインターネット接続が必要です。

ニュース見出しは投稿テーマの材料として使います。投稿前に、必ず元記事や一次情報を確認してください。

「話題ニュース」ジャンルでは、経営者、インフルエンサー、SNS炎上、トラブルなどの話題を取得しやすいキーワードを使います。実在人物に関する投稿は、事実確認をしたうえで、断定や中傷にならない表現に調整してください。

## コピーできる内容

画面上のボタンから以下をコピーできます。

- 各投稿文
- CSV形式テキスト
- 4コマ漫画作成プロンプト

確認用の簡易4コマPNGを作った場合は `outputs` フォルダに保存されます。

## 注意点

このツールは投稿文の下書きを作るためのものです。投資助言、売買指示、利益保証を行うものではありません。

実際に投稿する前に、内容が誤解を招かないか、投資助言になりすぎていないかを必ず確認してください。

このツール自体はOpenAI APIを使いません。4コマ漫画画像は、生成されたプロンプトをChatGPTの画像生成に貼り付けて作成してください。
